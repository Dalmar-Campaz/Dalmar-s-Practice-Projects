# Reproductor de Música Web

Este es un proyecto personal en desarrollo. Se trata de un reproductor de música creado con tecnologías full stack. El objetivo es practicar la conexión entre frontend y backend, y aplicar lo aprendido en HTML, CSS, JavaScript y Node.js.

---

## Estado del proyecto

- [x] Planificación del proyecto
- [ ] Desarrollo del frontend (HTML, CSS, JS)
- [ ] Backend básico (Node.js + Express)
- [ ] Subida y reproducción de archivos de audio
- [ ] Estilos y diseño responsive

---

## Tecnologías previstas

**Frontend:**

- HTML
- CSS
- JavaScript
- Font Awesome (para íconos)

**Backend:**

- Node.js
- Express.js

---

## Cómo empezar

1. Crear la estructura básica del frontend (`index.html`, `style.css`, `script.js`)
2. Construir los botones del reproductor con íconos (play, pausa, etc.)
3. Conectar con un servidor backend básico usando Express
